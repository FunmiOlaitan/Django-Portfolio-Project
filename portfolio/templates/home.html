My Projects

Welcome to my projects page! Here, you will find a collection of some of my work, showcasing the diverse range of skills and technologies I've employed to bring ideas to life.

Road Sign Detection using Machine Learning

The Road Sign Detection Project utilises deep learning techniques, specifically Convolutional Neural Networks (CNNs), alongside TensorFlow and Keras, to develop a robust model aimed at identifying traffic sign images. To increase the model's accuracy, data preprocessing and augmentation methods are applied. Additionally, visualisation tools are integrated into the project for data exploration. Overall, the traffic sign model achieved a training accuracy of approximately 80% and a validation accuracy of around 87%.

Smart Mirror with Tkinter

This project developed using Tkinter in Python, incorporates weather, news, and date/time features for a dynamic display. It fetches live weather data from the OpenWeatherMap API, showing temperature, description, and a weather icon. News headlines are retrieved from the NewsAPI. The date/time section presents the current time, date, and a personalised greeting. Designed for a Raspberry Pi connected to a display, it offers users an interactive experience.

Data Entry: Rock Climbing Club Membership Database

This is a Rock Climbing Club Membership Management System, utilising Python's Tkinter, SQLite, and Matplotlib. The system allows users to add, display, and clear memberships, with data validation ensuring accurate information entry. Scheduled backups are implemented to secure data integrity. Additionally, the project employs logging for tracking membership additions and clearing events. It also generates insights using SQL views and a visualisation of membership distribution.
